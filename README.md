## 🚀 How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/riturajkashyap70/data-analysis-projects.git
   ```

2. Navigate to any project folder:

   ```bash
   cd PROJECT_NAME
   ```

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Run analysis:

   ```bash
   jupyter notebook
   ```
